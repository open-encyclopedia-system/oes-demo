Class Versioning

This class hooks processing concerning the feature "Versioning" to post actions. "Versioning" includes a version controlling post type "Master" and a version controlled post type "Version".

The feature "Translating" is a special case of the feature "Versioning" where a post "Origin Post" is linked to a post "Translation" of the same post type. This feature can be combined with a master post type.
